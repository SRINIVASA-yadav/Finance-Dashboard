# Finance Dashboard

A simple, interactive finance dashboard built with React and JavaScript. Designed for beginners to learn frontend development basics.

## Features

- **Dashboard Overview**: Displays total balance, income, and expenses with summary cards.
- **Visualizations**: Balance trend line chart and spending breakdown pie chart using Chart.js.
- **Transactions Section**: List of transactions with search and filter functionality.
- **Role-Based UI**: Switch between Viewer (read-only) and Admin (can add transactions) roles.
- **Insights**: Shows highest spending category and income vs expenses comparison.
- **Responsive Design**: Works on different screen sizes.

## Technologies Used

- React 18 (with JSX)
- JavaScript (ES6+)
- Chart.js for visualizations
- HTML5
- CSS3 (Flexbox for layout)

## Setup Instructions

1. Clone or download the project files.
2. Open `index.html` in a web browser (Chrome, Firefox, etc.).
3. No server or build process required - it's a static site using CDN React.
4. You can also click this link in Markdown preview: [Open index.html](./index.html)

## Project Structure

- `index.html`: Main HTML file with React CDN scripts.
- `style.css`: Styles for responsive design and UI.
- `script.js`: React components and logic using JSX.
- `data.js`: Mock transaction data.
- `.github/copilot-instructions.md`: Project setup checklist.

## How to Use

1. Open `index.html` in your browser.
2. View the overview cards and charts.
3. Scroll to Transactions to see the list and use search/filter.
4. Switch role to Admin to add new transactions.
5. Check Insights for spending analysis.
